# html-color-picker
HTML Color Picker

A small website to convert between color coordinates
* HEX
* RGB (0-255)
* RGB (0-1)

Relies on color-convert

## Screenshots
![](README_Resources/screenshot1.png)


## Setup
* `git clone`
* `npm ci`
* `npx webpack`
